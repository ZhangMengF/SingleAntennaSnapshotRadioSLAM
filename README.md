# Codes for Single-Antenna Snapshot Radio-SLAM
## Project Structure

This repository contains two main sub-projects:

### 1. Radio_SLAM
This folder implements the SLAM algorithm and evaluates its performance.
*   **Environment**: MATLAB R2025b
*   **Function Script:
   *   'Base.m'
*   **Main Scripts**: 
    *   `main_Performance_DA.m`
    *   `main_Performance_LoSDetection_Map.m`
    *   'main_Performance_LoSDetection_Loca.m'

### 2. Raytracing
This folder simulates wireless channels using Sionna ray tracing.
*   **Language**: Python 3.13.7
*   **Library**: Sionna 1.2.0
*   **Main Script**: `main.py`
